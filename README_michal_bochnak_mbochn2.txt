CS 474 Project 02 - Assembly Language Interpreter
March 21, 2018

-----------------------------------------------------
Project was developed on Windows 10.
Command pattern implemented.

-----------------------------------------------------
Enter code into the box on the left and click
'Load code into RAM' after you are done.
One instruction per line.

Example:
LDI -1
XCH
LDI 1
ADD
DEC X
ST X
HLT
DEC T

'none' represents the memory that was not initialized.
If entered code is longer that 256 lines program will
load first 256 lines and discard the rest.
